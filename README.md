# CarritoDeCompras

Proyecto en Java que implementa un carrito de compras aplicando refactorización.

## Mejoras realizadas

- Separación de responsabilidades
- Encapsulación con getters y setters
- Validación de precios
- Manejo de excepciones
- Uso de constantes
- Implementación de cantidad de productos
- Mejora de escalabilidad con HashMap
