# CarritoDeCompras

Proyecto en Java que implementa un carrito de compras aplicando refactorización y buenas prácticas de programación.

## Mejoras aplicadas

- Separación de responsabilidades
- Encapsulación con getters y setters
- Manejo de excepciones
- Uso de constantes
- Eliminación de duplicación de código
- Uso de HashMap para mejorar escalabilidad
- Implementación de cantidad de productos
- Documentación con Javadoc

## Estructura

- Item.java → Representa un producto
- CarritoDeCompras.java → Lógica del carrito
- Main.java → Programa principal
